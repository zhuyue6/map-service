2D 地图应用 ​

基于2d地图服务的能力，根据业务需求封装的可扩展的地图应用程序。除了基本模块下，其余进行插件化增强，可以达到高定制化的能力, 目前有 基础模块、元素插件、工具插件等功能。
地图应用与服务可能有一些重名的定义情况，如应用元素Element与服务元素SElement，这里命名前缀多一个S来代表使用的是服务定义相关

  • 基础模块

    1. 底图
      同地图服务container baseMap, 底图设置
    2. 视图
      同地图服务view, 控制地图的视窗如何展示
    3. 事件反射器 应用事件相应回调
    4. 插件注入 提供应用插件注入的能力

  • 元素插件

    1. 基础元素插件
      提供元素的基本操作入口：增删改查等
    2. ap
      提供ap 元素的操作

  • 工具插件

    1. 基础工具插件
      提供工具的入口
    2. 移动
      提供元素移动的功能
    3. 选择
      提供选择元素的功能
    4. 修改
      提供修改元素的功能
    5. 测量
      提供测绘的功能
    6. 编辑
      提供编辑元素的功能
    7. 绘制
      提供绘制元素的功能
