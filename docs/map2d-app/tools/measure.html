measure ​

measure 提供测绘能力，包括测面积和测距离，通过measure反射回调

ts
  const { measure } = tools
  measure.enable()

  type MeasureType = 'distance' | 'area'

事件

事件参数是否必填描述
enableundefined否启用
closeundefined否关闭
useMeasureType是切换测量类型

反射

属性参数描述
measureElement[]测量结束后的反射事件
ts
  interface Measure {
    type: 'distance' | 'area'
    data?: number
  }
  app.emitter.on('measure', (measure: Measure)=>{
    console.log(measure)
  })
